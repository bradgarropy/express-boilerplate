# Feedback

## Submit Feedback

Send feedback to the site administrators.

```
POST /feedback
```

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | :------: | ----------- | ------- |
| feedback | string | yes | User feedback. | I love this site! |

### Response

```
{
    "message": "Thank you for your feedback!"
}
```
